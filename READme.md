The project is a product of the book Kern_and_Ritch, and the lessons are many of the challenges and illustrations dipicted from the book. The book is a great source of information on C programming language, the code snippets have well commented notes that's meant for the reader. Different levels of programmers, from novice to professional should be able to use this material, both as referencing or as a refresher from topics that are of the interest to the reader.


The project assumes that you are a first time programmer and thus for the beginners, feel free to skip through the lessons and find whats of your interest. The lessons on this project have been named after their prog unlike labelling them with numbers so they are easily readable.
